Robert Heaton compares the wave function collapse algorithm to creating a seating chart at a wedding.

Videos

Coding Resources for Beginners

There are several websites that provide tutorials for beginners. Dan Shiffman, who teaches at the Interactive Telecommunications Program at NYU's Tisch School for the Arts has some wonderful tutorials for beginners on his Coding Train website.

The Code Guppy is a site where kids 13+ and teens can learn JavaScript coding through fun and easy to follow tutorials.
